# ShelterAI

ShelterAI is an initiative designed to improve the allocation of places in reception centers for refugees and asylum seekers. The project's primary goal is to prioritize and place the most vulnerable individuals fairly and efficiently, reducing wait times and optimizing resource usage for host organizations.

Main Objectives:
- Prioritize the most vulnerable individuals and families.
- Improve efficiency in the allocation of available spaces.
- Provide tools that support decision-making for NGOs and management teams.

Key Features:
- Vulnerability assessment to prioritize care.
- Automatic allocation of places based on needs and availability.
- Support for analysis and simulations to assist in capacity planning.
